* Why Nibelung?
Nibelung Theme is a minimalist Emacs color scheme blending cool gray tones and subtle blue accents, reminiscent of the Nebelung cat's silky fur. Designed for developers who value clarity and focus during long coding sessions.
#+begin_quote
3 AM. Brain fog. Eye strain. One thing remains sharp: your goddamn readable code.
#+end_quote
* Who is it for?
- Fans of ~doom-flatwhite~ and ~doom-plain~ seeking a fresh minimalist twist
- Developers working for hours: soft contrasts reduce eye strain
- ~Nord~ and ~gruvbox~ enthusiasts ready for a more "airy" alternative
* Features
- Light and dark variants from a single palette
- Comments become first-class citizens, not afterthoughts
- 40+ packages with explicit face definitions
  + Completion: Company, Corfu
  + Navigation: Vertico, Consult, Avy, Which-key
  + Git: Magit, Diff-HL, Git Gutter
  + Files: Dired, Diredfl, Dirvish, Treemacs
  + LSP: Eglot (with semantic tokens), LSP-UI
  + Syntax: Tree-sitter, Flycheck, Flymake
  + Org Mode, Markdown
  + Modeline: Doom Modeline, Doom Nano Modeline
  + Evil: Evil Goggles, Evil Snipe
  + Debug: Dape, Cider, EJC
  + UI: Hydra, Transient, Rainbow Delimiters, Solaire

** Emacs
*** straight.el
#+begin_src emacs-lisp
(straight-use-package
 '(nibelung-theme :type git :host github :repo "veschin/nibelung-theme"
                  :files ("*.el")))
(load-theme 'nibelung t)      ; light
(load-theme 'nibelung-dark t) ; dark
#+end_src
*** Doom Emacs
#+begin_src emacs-lisp
;; packages.el
(package! nibelung-theme
  :recipe (:host github
           :repo "veschin/nibelung-theme"
           :files ("*.el")))
;; config.el
(setq doom-theme 'nibelung)      ; light
(setq doom-theme 'nibelung-dark) ; dark
#+end_src
*** package.el
#+begin_src emacs-lisp
(use-package nibelung-theme
  :vc ( :url "https://github.com/veschin/nibelung-theme"
	:rev :newest)
  :ensure t
  :config
  (load-theme 'nibelung t))
#+end_src
*** Warning
U should disable any other theme
** VSCode
#+begin_src sh
git clone https://github.com/veschin/nibelung-theme.git /tmp/nibelung && \
  cp -r /tmp/nibelung/dist/vscode ~/.vscode/extensions/nibelung-theme
#+end_src
~Ctrl+K Ctrl+T~ -> select "Nibelung" or "Nibelung Dark"
** Neovim
#+begin_src sh
git clone https://github.com/veschin/nibelung-theme.git /tmp/nibelung && \
  mkdir -p ~/.config/nvim/colors && \
  cp /tmp/nibelung/dist/neovim/colors/*.lua ~/.config/nvim/colors/
#+end_src
#+begin_src lua
-- init.lua
vim.cmd("colorscheme nibelung")      -- light
vim.cmd("colorscheme nibelung-dark") -- dark
#+end_src
** IntelliJ IDEA
#+begin_src sh
git clone https://github.com/veschin/nibelung-theme.git /tmp/nibelung && \
  cp /tmp/nibelung/dist/intellij/*.icls ~/.config/JetBrains/*/colors/
#+end_src
Settings -> Editor -> Color Scheme -> select "Nibelung" or "Nibelung Dark"
** Alacritty
#+begin_src sh
curl -o ~/.config/alacritty/nibelung.toml \
  https://raw.githubusercontent.com/veschin/nibelung-theme/main/dist/alacritty/nibelung.toml
curl -o ~/.config/alacritty/nibelung-dark.toml \
  https://raw.githubusercontent.com/veschin/nibelung-theme/main/dist/alacritty/nibelung-dark.toml
#+end_src
#+begin_src toml
# alacritty.toml
[general]
import = ["~/.config/alacritty/nibelung.toml"]      # light
import = ["~/.config/alacritty/nibelung-dark.toml"]  # dark
#+end_src
** Telegram Desktop
#+begin_src sh
git clone https://github.com/veschin/nibelung-theme.git /tmp/nibelung && \
  cp /tmp/nibelung/dist/telegram/*.tdesktop-palette ~/
#+end_src
Send the ~.tdesktop-palette~ file to Saved Messages in Telegram, then tap to apply. Apply light in day mode, dark in night mode -- Telegram auto-switches.
** Caelestia / Quickshell
#+begin_src sh
git clone https://github.com/veschin/nibelung-theme.git /tmp/nibelung && \
  cp /tmp/nibelung/dist/caelestia/nibelung-dark-scheme.json \
  ~/.local/state/caelestia/scheme.json
#+end_src
For light variant use ~nibelung-scheme.json~. Caelestia hot-reloads the scheme.
** OpenCode
#+begin_src sh
mkdir -p ~/.config/opencode/themes && \
  curl -o ~/.config/opencode/themes/nibelung.json \
  https://raw.githubusercontent.com/veschin/nibelung-theme/main/dist/opencode/nibelung.json
#+end_src
#+begin_src json

{ "theme": "nibelung" }

Both light and dark variants included. OpenCode selects based on terminal background.
